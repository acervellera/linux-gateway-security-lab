# Report locali

Questa cartella è riservata ai report generati durante il laboratorio.

Il file `.gitignore` esclude il contenuto della cartella perché i report possono contenere indirizzi, nomi di interfaccia, domini e altre informazioni della rete locale.

Prima di condividere un report:

1. crearne una copia;
2. rimuovere dati sensibili;
3. ridurlo alle sole righe necessarie;
4. spostare l'eventuale esempio revisionato in `samples/`;
5. non pubblicare mai il file originale senza controllo.
