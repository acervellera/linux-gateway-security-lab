# Esempi anonimizzati

Questa cartella conterrà piccoli campioni di dati usati per imparare e testare il codice Python.

Sono ammessi soltanto esempi:

- prodotti da dispositivi propri o autorizzati;
- ridotti al minimo necessario;
- revisionati manualmente;
- privi di password, cookie e contenuti personali;
- con IP, MAC, hostname, domini e SSID anonimizzati quando necessario.

Non pubblicare catture PCAP grezze. Preferire poche righe JSON o tabelle create appositamente per i test.
