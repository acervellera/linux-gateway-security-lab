# Script di supporto

Questa cartella conterrà script Bash usati per inventario, verifica, backup e raccolta controllata di dati.

Gli script devono essere:

- commentati;
- non distruttivi per impostazione predefinita;
- espliciti quando richiedono `sudo`;
- dotati di controllo degli argomenti;
- dotati di messaggi di errore comprensibili;
- accompagnati da esempi e rollback quando modificano il sistema.

Gli script verranno aggiunti solo dopo aver eseguito manualmente e compreso i comandi che automatizzano.
