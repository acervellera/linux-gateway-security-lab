# Python

Questa cartella conterrà il codice Python sviluppato durante il progetto.

Il percorso inizierà con piccoli esercizi e arriverà a un analizzatore strutturato dei log Suricata e Zeek.

Ogni file dovrà spiegare:

- a cosa servono le librerie importate;
- quali dati riceve;
- quali dati restituisce;
- come funziona ogni blocco importante;
- quali errori gestisce;
- come eseguire un test.

Non verrà aggiunto codice complesso prima di aver compreso e verificato la versione semplice.
